Supported Systems

apcupsd is designed to be able to run on any Posix system.

Supported Systems

apcupsd has been built and tested on:
  • RedHat
  • Solaris
  • FreeBSD
In addition, apcupsd has been ported to the following systems:
  • Alpha
  • Darwin
  • Debian
  • Engard
  • Gentoo
  • HP/UX
  • Mandrake
  • NetBSD
  • OpenBSD
  • Slackware
  • SuSE
  • Unifix
  • Win95/98/Me/XP/NT/2000
